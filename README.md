# Simulador de Investimnetos

Projeto simples em Python para simular o crescimento de um investimento com juros compostos.

## Funcionalidades
- Receber valor inicial 
- Recebe taxa de juros anual 
- Recebe tempo em anos 
- Calcula valor final 
- Calcula lucro obtido

## Tecnologias 
- Python3 

## Como executar

```bash
python3 simulador.py
